## ðŸ“‹ Submission Rules

### â° Deadline
- **Hard deadline:** Sunday, June 15, 2025 at 16:00 CEST
- **No late submissions accepted**

### ðŸ“ Format Requirements
- **One project per Pull Request**

### ðŸ› ï¸ Technical Rules

- **Chain-agnostic:** No restrictions on blockchain platforms

### ðŸ‘¥ Team Rules
